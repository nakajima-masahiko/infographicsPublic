💻 コード実行
💻

1. Code Execution

code_execution

ステートフルなコードインタープリタで、コードを実行して出力を確認できます。 REPL環境なので、前の実行結果が保持されます。アタッチメントのファイルにアクセス可能で、 ファイル名を直接コードで参照します（例: open('test.txt', 'r')）。 デフォルトのPython 3.12.3環境で、基本ライブラリやSTEM関連ライブラリ（numpy, sympy, rdkitなど）が利用可能。 インターネットアクセスはpolygonとcoingeckoのみで、追加パッケージのインストールは不可。

📝 引数
code string 必須
実行するコード
🌐 Web検索・閲覧
🗏

2. Browse Page

browse_page

指定したウェブサイトのURLからコンテンツを取得します。 LLMサマライザーを使用して、提供された指示に基づいて抽出/要約します。

📝 引数
url string 必須
閲覧するウェブページのURL
instructions string 必須
サマライザーへの指示。明確で自己完結したものにし、広範な概要や特定の詳細を指定
🔍

3. Web Search

web_search

ウェブを検索します。site:reddit.comなどの検索演算子を使用可能。

📝 引数
query string 必須
検索クエリ
num_results integer オプション
結果数（デフォルト10、最大30）
𝕏 X（旧Twitter）連携
🔎

4. X Keyword Search

x_keyword_search

Xの投稿を高度に検索するツール。

📝 引数
query string 必須
Xの高度検索クエリ文字列。すべての高度演算子をサポート（例: (puppy OR kitten) filter:images）
limit integer オプション
返却する投稿数（デフォルト3）
mode string オプション
ソートモード：TopまたはLatest（デフォルトTop）
🔭

5. X Semantic Search

x_semantic_search

意味検索クエリに関連するXの投稿を取得。

📝 引数
query string 必須
意味検索クエリ
limit integer オプション
返却する投稿数（デフォルト3）
from_date string | null オプション
開始日（YYYY-MM-DD、デフォルトNone）
to_date string | null オプション
終了日（YYYY-MM-DD、デフォルトNone）
exclude_usernames array | null オプション
除外ユーザー名（デフォルトNone）
usernames array | null オプション
含むユーザー名（デフォルトNone）
min_score_threshold number オプション
最小関連スコア（デフォルト0.18）
👤

6. X User Search

x_user_search

検索クエリに基づいてXユーザーを検索。

📝 引数
query string 必須
検索する名前またはアカウント
count integer オプション
返却するユーザー数（デフォルト3）
💬

7. X Thread Fetch

x_thread_fetch

X投稿とそのコンテキスト（親やリプライ）を取得。

📝 引数
post_id integer 必須
投稿ID

8. View Image

view_image

指定したURLまたはIDの画像を表示。

📝 引数
image_url string | null オプション
画像URL（デフォルトNone）
image_id integer | null オプション
画像ID（デフォルトNone）

9. View X Video

view_x_video

X上のビデオのフレームと字幕を表示。URLはXホストのビデオに直接リンク。

📝 引数
video_url string 必須
ビデオURL
🔍

12. Search Images

search_images

説明に基づいて画像を検索。視覚的な文脈を追加する場合に使用。

📝 引数
image_description string 必須
画像の説明
number_of_images integer オプション
画像数（デフォルト3）
📄 PDF操作

10. Search Pdf Attachment

search_pdf_attachment

PDFファイルで検索クエリに関連するページを検索。ページ番号とテキストスニペットを返却。

📝 引数
file_name string 必須
PDFファイル名
query string 必須
検索クエリ
mode string 必須
検索モード：keywordまたはregex
🗎

11. Browse Pdf Attachment

browse_pdf_attachment

PDFファイルを閲覧。指定ページのテキストとスクリーンショットを返却。

📝 引数
file_name string 必須
PDFファイル名
pages string 必須
ページ番号（カンマ区切り、1-indexed、例: '1,3,5-7'）
💬 会話履歴
💭

📝 引数
query string 必須
意味検索クエリ
