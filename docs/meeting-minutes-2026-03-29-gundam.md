# 議事録

- 日時: 2026-03-29
- 形式: 定例戦略ミーティング
- 参加者: ブライト、クアトロ、カミーユ、フォウ、ハマーン
- 書記: Codex

## 1. 目的

各勢力間の緊張が続く状況において、短期的な衝突回避と中長期の協調可能性を検討する。

## 2. 議題

1. 現状認識の共有
2. 民間被害を抑える行動原則
3. 指揮系統と情報連携
4. 今後1か月の優先アクション

## 3. 発言要旨

### ブライト

- 現場指揮の観点から、単独判断による先制行動は全面的に抑制すべき。
- 戦闘回避のため、交戦規定を明文化し全員で順守する必要がある。

### クアトロ

- 対話の窓口を維持し続けることが、結果的に最小の損失につながる。
- 強硬策をちらつかせるだけではなく、譲歩可能な条件を事前に用意すべき。

### カミーユ

- 現場の心理的負荷が限界に近く、意思決定の遅れが二次被害を生む。
- 若年パイロットのケア体制と、判断を支えるメンター配置を提案。

### フォウ

- 当事者の記憶や感情を軽視した交渉は破綻しやすい。
- 人道面の合意（避難回廊・医療搬送）を先に固定するべき。

### ハマーン

- 力の均衡が崩れると合意は一瞬で無効化されるため、抑止力の設計が必要。
- ただし、相互に破滅を避ける最低限のルールには同意可能。

## 4. 合意事項

1. 7日以内に「非戦闘員保護プロトコル」草案を作成（責任者: ブライト）。
2. 3勢力間の連絡窓口を一本化し、誤認交戦防止のホットラインを設定（責任者: クアトロ）。
3. メンタルケア担当を任命し、現場ヒアリングを週次で実施（責任者: カミーユ）。
4. 医療・避難に関する中立運用ルールの叩き台を作成（責任者: フォウ）。
5. 抑止バランス評価レポートを次回会議までに提出（責任者: ハマーン）。

## 5. 保留事項

- 各勢力の戦力配置情報の開示範囲
- 共同声明の文言（「停戦」か「限定的衝突回避」か）
