ユースケース 01 — Code Generation

コード生成・実装支援

「何を作りたいか」を伝えるだけで、動くコードの雛形を即座に生成。
コンテキスト（言語・フレームワーク・既存コードの制約）を与えるほど精度が上がる。

📝 プロンプト例
TypeScript + Express で、
JWT認証ミドルウェアを作って。
・アクセストークンは Authorization: Bearer
・期限切れは401、無効は403を返す
・ユーザー情報は req.user に格納
✅ 生成されるコード（抜粋）
import { Request, Response, NextFunction } from 'express'; import * as jwt from 'jsonwebtoken'; export const authMiddleware = ( req: Request, res: Response, next: NextFunction ) => { const token = req.headers.authorization ?.split(' ')[1]; if (!token) return res.status(401).json({...}); try { req.user = jwt.verify(token, process.env.JWT_SECRET); next(); } catch (e) { res.status(403).json({ error: 'Invalid token' }); } };
🎯 コンテキストを渡す
言語・フレームワーク・既存コードの制約を最初に伝える
🔄 反復で精度を上げる
「エラーハンドリングを追加して」「テストも書いて」と続けて依頼
🧪 必ず動作確認
生成コードはそのままコピペせず、ロジックをレビューしてから使う
📁 既存コードを貼る
「このコードに合わせて」と既存ファイルを貼ると一貫性が増す
