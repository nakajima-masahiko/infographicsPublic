Copilot エージェント機能 一覧
🤖
Coding Agent
GA 2025

Issue を自律的に解決し draft PR を作成。GitHub Actions で動く安全な環境で、コード・テスト・デバッグまで完全自動化。

  • Issue コンテキスト・画像を理解
  • インクリメンタルにコミットを Push
  • レビューコメントに自動対応
  • 68.6% の PR マージ成功率
🖥️
Agent Mode (IDE)
GA 2025

VS Code / JetBrains IDE 上で動く自律型エージェント。複数ファイルを同時編集し、エラーを自己修正しながら完了まで走り続ける。

  • サブタスクを自動識別・実行
  • ターミナルコマンドの自律実行
  • ランタイムエラーの自己修正
  • テスト失敗の自動分析・修正
👁️
Agentic Code Review
2026年3月

PR 全体のコンテキストを把握した上でコードレビューを実施。提案を Coding Agent に渡して修正 PR を自動生成できる。

  • プロジェクト全体を考慮したレビュー
  • インラインコメントとして提案
  • 修正を Coding Agent へ自動委託
  • VS Code / JetBrains 対応
🔗
MCP サポート
2025-2026

リポジトリの設定で MCP サーバーを設定し、エージェントが外部データ・ツールにアクセス可能に。

  • リポジトリ設定で MCP サーバーを構成
  • GitHub ローカル MCP でリポジトリ操作
  • JetBrains では自動承認サポート
  • Slack・Jira・DB 等と連携可能
🏢
Agent HQ
GitHub Universe 2025

複数エージェントをソフトウェアライフサイクル全体にわたって作成・管理・調整するための中央ワークスペース。

  • エージェントの一元管理ダッシュボード
  • ライフサイクル全体をカバー
  • エージェント間の協調をサポート
🌊
Agentic Workflows (CI/CD)
Technical Preview 2026

YAML の代わりに Markdown でワークフローを記述。AI が条件付きタスクを解釈して CI/CD パイプラインを自律的に実行。

  • Markdown でワークフロー定義
  • AI が条件・分岐を自律判断
  • 既存 Actions との互換性維持

Squad — オープンソースのリポジトリネイティブ・マルチエージェント

Squad はリポジトリ内でコーディネーターエージェントが専門エージェントを起動する OSS フレームワーク。重厚な中央集権インフラ不要で GitHub ネイティブなマルチエージェントチームを構成できます。コーディネーターがタスクをルーティングし、PM・Dev・Test・Review の各専門エージェントが並列作業します。
