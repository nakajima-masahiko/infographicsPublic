GitHub Copilot Coding Agent とは

GitHub Copilot Coding Agent は、GitHub Actions が動力源の 自律型 AI ソフトウェアエンジニア です。 GitHub Issues やCopilot Chatでタスクを割り当てると、クラウド上のエフェメラルな開発環境で コードを読み・書き・テスト・レビューし、最終的に Pull Request を作成 して人間のレビューを依頼します。 IDE で使う「エージェントモード」とは異なり、完全に非同期・バックグラウンドで動作します。
