pandas & openpyxl

データ操作・分析の定番ライブラリと、Excelファイルを自在に操るライブラリ。
何ができるかをビジュアルでわかりやすく解説します。

🐼

pandas

v2.x / 2024+

表形式データ（DataFrame）を扱うための強力なライブラリ。 CSV・Excel・JSONなど多様な形式の読み込み・書き出しに対応し、 フィルタリング・集計・結合・可視化まで一貫して処理できます。 データサイエンス・機械学習の前処理に不可欠な存在です。

📊

openpyxl

v3.x / 2024+

Python から Excel（.xlsx / .xlsm）ファイルを読み書きする専用ライブラリ。 セルの書式・色・フォント設定、グラフ挿入、数式記入、 シートの追加・削除など、Excel操作をコードで完全自動化できます。

📦 インストール

# pip でインストール
pip install pandas openpyxl

# conda でインストール
conda install pandas openpyxl

🐼 pandas でできること

📥

多形式データの読み込み

CSV・Excel・JSON・SQL・Parquet・HTML など多様なフォーマットを一発で DataFrame に変換。pd.read_csv() などひと行で済みます。

🔍

フィルタリング・抽出

条件式でデータを絞り込んだり、特定の列・行を取り出したり。df[df['売上'] > 1000] のような直感的な構文で操作できます。

🧹

データクレンジング

欠損値（NaN）の検出・補完・削除、重複行の除去、データ型変換、文字列の正規化など、データ前処理の全工程をサポートします。

📊

集計・グループ化

groupby() でカテゴリ別に集計、pivot_table() でクロス集計。合計・平均・最大最小・カウントを一括計算できます。

🔗

テーブルの結合・マージ

merge() でSQL的な INNER / LEFT / RIGHT / OUTER JOIN、concat() で縦横に連結。複数データを自在に組み合わせます。

📈

時系列データの処理

日付インデックス・リサンプリング（日次→月次変換）・移動平均・ラグ計算など、時系列固有の処理を豊富なAPIで実現します。

🎨

可視化（Matplotlib連携）

df.plot() ひと行で折れ線・棒・ヒストグラム・散布図を生成。Matplotlib・Seaborn・Plotlyとのシームレスな連携も可能です。

📤

多形式への書き出し

処理後のデータをCSV・Excel・JSON・Parquet・SQLなど好みの形式で保存。to_excel() で Excel ファイルへのエクスポートも簡単です。

📊 openpyxl でできること

📖

Excel ファイルの読み込み

既存の .xlsx / .xlsm ファイルを開き、シート・セルの値・数式・書式情報をPythonオブジェクトとして取得できます。

✏️

セルへの書き込み・編集

特定のセルに値・数式・文字列を書き込みます。ws['A1'] = "売上合計" のように直感的に操作できます。

🎨

セルの書式・スタイル設定

フォントの種類・サイズ・太字・色、セルの背景色、罫線スタイル、数値フォーマット（通貨・%表示など）を細かく設定できます。

📑

シートの追加・複製・削除

新しいワークシートの追加、既存シートの名前変更・複製・削除、シートの表示順変更をコードで自動化できます。

📐

行・列の操作

行・列の挿入・削除、列幅・行高さの設定、セルの結合（merge_cells）・分割、フリーズペインの設定ができます。

📉

グラフの挿入

棒グラフ・折れ線グラフ・円グラフ・散布図など多種類のグラフをPythonコードで生成し、シートに埋め込めます。

🖼️

画像の挿入

PNGやJPEGなどの画像ファイルをシートの任意の位置に挿入できます。ロゴ・ヘッダー画像の自動挿入に便利です。

✅

データバリデーション

ドロップダウンリスト、数値範囲の制限、日付制限などの入力規則（バリデーション）をセル範囲に設定できます。

💻 コードサンプル

🐼 pandas — CSVを読んで集計・出力
import pandas as pd

# CSVファイルを読み込む
df = pd.read_csv("sales.csv")

# 欠損値を除去
df = df.dropna()

# 売上が1000以上の行を絞り込む
filtered = df[df["売上"] >= 1000]

# 月別・商品別に集計
summary = filtered.groupby(
    ["月", "商品"]
)["売上"].sum().reset_index()

# Excelファイルとして書き出す
summary.to_excel(
    "output.xlsx",
    index=False
)
📊 openpyxl — Excel整形・グラフ挿入
from openpyxl import load_workbook
from openpyxl.styles import Font, PatternFill
from openpyxl.chart import BarChart, Reference

wb = load_workbook("output.xlsx")
ws = wb.active

# ヘッダーを太字・背景色付きに
for cell in ws[1]:
    cell.font = Font(bold=True, color="FFFFFF")
    cell.fill = PatternFill(
        fill_type="solid",
        fgColor="0284C7"
    )

# 棒グラフを生成して挿入
chart = BarChart()
data = Reference(ws, min_col=3,
    min_row=1, max_row=ws.max_row)
chart.add_data(data, titles_from_data=True)
ws.add_chart(chart, "E2")

wb.save("report.xlsx")
